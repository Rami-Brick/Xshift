# Xshift Handoff Documentation

This folder documents the Xshift employee attendance PWA being built in this repository (`XshiftClaude`).

Read these files in order:

1. `PROJECT_BRIEF.md` - product goals and scope
2. `ARCHITECTURE.md` - stack, folder layout, auth strategy
3. `DATABASE_SCHEMA.md` - tables, enums, RLS policies
4. `WORKFLOWS.md` - feature-level product behavior
5. `IMPLEMENTATION_PLAN.md` - phased build plan
6. `SETUP.md` - one-time Supabase + Vercel bootstrap
7. `MANAGER_PUSH_NOTIFICATIONS_PLAN.md` - Web Push notification plan for manager/admin attendance alerts

The single source of truth for execution order is `IMPLEMENTATION_PLAN.md`.
